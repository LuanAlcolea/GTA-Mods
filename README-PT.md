# GTA Mods
## Para que este repositório existe?
Este repositório guarda os arquivos de mods feitos para o jogo GTA 3. Os mods são scripts CLEO que podem ser executados com o CLEO instalado no seu GTA, pode ser instalado pelo Modloader. Os scripts são escritos por mim, há alguns que são adaptados de outros autores, os devidos créditos são explicitados nos arquivos.
Não vou abordar como instalar os mods aqui, a internet está repleta de tutoriais de como instalar mods com o CLEO/Modloader etc.
## Mods para o GTA 3
## Pular do carro
Script CLEO que permite pular do carro assim como os jogos posteriores, para usar basta apertar o botão de sair do carro enquanto o veículo está maior que a velocidade mínima.

https://forum.mixmods.com.br/f5-scripts-codigos/t5530-iii-pular-do-carro

## Ligar/Desligar motor do carro
Script CLEO que permite desligar o motor do carro ao apertar segurado o botão de sair. O funcionamento imita esse comportamento de GTAs posteriores para este GTA.

https://forum.mixmods.com.br/f5-scripts-codigos/t6942-iii-iv-engine-on-off

## Ativar hidráulicas em qualquer carro
Script CLEO simples que permite o jogador ativar as hidráulicas do carro ao digitar uma sequencia de palavras.

https://forum.mixmods.com.br/f5-scripts-codigos/t6714-iii-hydraulics-in-any-car

## Zoom no radar
Script CLEO que permite o zoom no radar e mostra o nome da rua e do carro na tela. Este script foi adaptado para o GTA 3, ou seja, originalmente foi escrito pelo "spaceeinstein" para o GTA VC, eu li o script original e refiz para o GTA 3.

https://forum.mixmods.com.br/f5-scripts-codigos/t6689-iii-radarzoom

## No Clip
Este é um No Clip para o GTA 3 inspirado pelo No Clip do GTA SA feito pelo "junior_djjr", entretanto devido ao mod não estar completo ele apresenta alguns bugs, mas é muito bem utilizável.

*Source está neste repositório*

## Restore Icons
Este script permite visualizar ícones de curtos alcances no radar do jogo. Os ícones visíveis são de Pay 'n' Spray, Ammu-Nation e  8-Balls Bomb Shop, aparecem quando a posição de cada uma dessas localizações estão dentro do radar e desaparecem quando estão fora do radar.

*Souce está neste repositório*
